🔐 Reverse Shell Security Lab
📌 Overview

This project demonstrates a reverse shell in a controlled lab environment using Android (Termux) and Ubuntu systems. It shows how a target machine connects back to an attacker for remote command execution.

⚙️ Tools Used
Python
Kali Linux
Ubuntu
Termux
Netcat
📂 Project Structure
scripts/ → Python reverse shell scripts
docs/ → Technical report and working screenshots
🔁 How It Works
Attacker starts a listener
Target runs the script
Target connects back to attacker
Attacker executes commands remotely
